단어 : 단어로 입력하는 방식
컬러코드 : #웹 번호로 입력하는 방식
rgb : red, green, blue의 숫자 값을 입력하는 방식
rgba : rgb 표현 방식에 alphae 값 추가
hsl : hue, saturation, lightness의 양값을 입력하는 방식
hsla : hsl 표현 방식에 aplha 값 추가

* aplha
> 색상의 투명도를 결정

* opacity
> 상속의 특징을 가지고 있음
> 자식요소를 포함한 모든 부분의 투명도가 적용
> %, 소수점
> opacity : 0으로 요소를 투명하게 처리

* display : none
> 요소 자체를 삭제

box

* box-shadow
> 박스에 그림자를 넣는 속성 : x축 y축 그림자크기(블러값) 그림자색상
> 0px 0px 10px #333 > 좌우 크기를 0으로 주게되면 상하좌우 모두 넣을 수 있다
> rgba 값을 사용해 투명도 조절 가능

* text-shadow
> 텍스트에 그림자를 넣는 속성 : x축 y축 그림자크기(블러값) 그림자색상
