배경관련속성

* background-color
> 배경색 : 단어, hex, rgb..
> transparent : 투명
> background-image와 함께 사용할 수 있음

* background-img
> 배경 이미지를 불러오는 속성
> url() : 이미지의 경로
> 반복되는 성격
> ,(쉼표)로 구분지어 배경 이미지를 2장 이상의 이미지를 연결할 수 있음
> 먼저 작성한 순서대로 이미지가 상단에 표현
> png파일의 이미지를 사용해야 겹쳐 보일 수 있음

* background-repeat
> 배경 이미지의 반복 결정
> 기본값은 repeat
> no-repeat : 반복되지 않은 원본 한 장
> repeat-x : 가로 반복 (수평 상에서의 반복)
> repeat-y : 세로 반복 (수직 상에서의 반복)

* background-size
> 배경 이미지 크기 표현시 : px, %, cover, contain
> 픽셀 표현시 가로 세로 크기 입력 (이미지 비율 깨짐)
> % : 가로 상의 비율에 맞춰서 높이 값도 자동으로 맞춰짐
> cover : 화면상 꽉차게 표현, 가로세로 비율 최대한 유지
>> 더이상 줄어들 수 없을 때 이미지가 잘리면서 표현됨
> contain : 100%와 거의 비슷, 요소 내에서 원본 이미지의 비율을 유지하면서 표현하되 넘치지 않음

* background-attachment
> 배경 이미지를 어떤 방식으로 화면에서 표현할 지 지정하는 속성
> scroll : 스크롤바와 함께 배경 이미지 움직임 (기본값)
> fixed : 스크롤바가 움직여도 배경이미지 고정

* background-position
> 배경 이미지의 위치 지정시 - 폭값이 꼭 존재해야함
> x, y : 단어, px, %..
> 단어 : x - left, center, right / y - top, center, bottom

* background-clip
> 배경이 테두리, 패딩, 글자, 어느 영역까지 확장 되는 지 결정
> border-box : 기본 값
> text : 글자 안에 이미지 넣기 (글자 색은 transparent로 지정해야 함)

* 배경 그라디언트 표현식
> linear-gradient() : 선형 그라이디언트
> 기본적으로 수직으표 색상 표현
> 쉼표로 색상 구분해서 작성
> deg : 각도 표현
> to left, right, top, bottom
>> 위아래 방향을 정하고 좌우 방향 정해서 입력
> 색상 뒤에 퍼센트를 작성하면 그 값 만큼 색상이 차지
>> 퍼센트를 사용해서 완전히 직선으로 만들 수 있음
> CSS Gradient : 그라디언트를 미리 만들어서 코드를 가져올 수 있는 사이트
