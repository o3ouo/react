** display : 요소의 성질을 결정하는 속성 **

* block
- 박스와 같이 쌓이는 성격
- 모든 css 적용
- 부모 전체의 폭을 사용하는 성격
block1
block2
block3
* inline
- 글자와 같이 나열되는 성격
- 내용물에 따라 폭이 결정되는 성격
- 크기 + 마진 상하 부여 x
inline1 inline2 inline3
* display: inline;
- 요소 성질을 인라인 형태로 변경
divBox1
divBox2
divBox3
* display: block;
- 요소 성질을 블럭 형태로 변경
spanBox1 spanBox2 spanBox3
* display: inline-block;
- 인라인과 블럭의 합성 속성 글자와 같이 옆으로 나열되며 블럭과 같이 모든 css 적용이 이루어지게 됨
* none
- 요소를 제외
- 스크립트와 같이 많이 사용

* img { display: block; }
- 이미지는 인라인 블럭임
- 이미지라는 사이즈가 존재하기 때문
- 이미지 태그를 사용하면 꽉차있는 것이 아니라 아래에 보이지 않는 여백이 있어서 뜨게 된다
- 그래서 기본 값에 이미지 자체를 디스클레이 블럭 처리하고 사용
ㄴ 여백 값이 사라져서 이미지 사이즈 만큼만 가지면서 쌓이게 된다

* figure { width: fit-content; height: fit-content; }
- 박스 안에 이미지를 담아서 사용하면 이미지 만큼의 크기를 가지게 따로 지정을 해야 하는데
- 그것을 일일히 지정하지 않고 width, height 값에 fit-content 를 사용하면 알아서 이미지 만큼의 사이즈를 가지기 때문에 편리해서 사용함
