* 속성선택자
- 특정한 속성이나 특정한 속성값을 가지고 있는 html 요소를 선택 시
- [속성이름 = "속성값"]
> 속성 이름을 제시하고 일반적인 속성을 불러 값을 설정할 때
- [속성이름$ = "속성값"]
> 특정한 속성명이 존재할 때 속성명의 접미사로 끝나게 되는 속성 값을 설정할 때 ex. 확장자
- [속성이름* = "속성값"]
> 특정 문자가 들어간 속성값을 불러올 때


* input
> 속성에 대한 값에 따라 타입이 완전히 달라지게 된다
> type="text" 기본 값
> 패스워드 : 암호화 처리가 되어 입력된다

* :focus
> 요소에 초점이 맞춰진 경우 사용할 수 있음


* 문자선택자
- ::first-letter > 특정한 선택자에 첫 번째 글자를 선택
- ::first-line > 특정한 선택자에 첫 줄을 선택
