* 후손
- 해당 요소의 하위 요소에 있는 특정한 요소를 모두 선택하는 선택자
- 선택자a 선택자b (띄어쓰기로 구분)
- 위에서부터 언급해서 사용 (코드의 우선 순위)
* 자식
- 해당 요소의 바로 밑에 위치하는(하위) 특정한 요소를 모두 선택하는 선택자
- 선택자a > 선택자b (기호로 구분)

- 동위관계에 있는 요소 중에 해당 요소보다 뒤에 존재하는 특정한 요소를 선택할 때 사용
- 선택자a + 선택자b : 선택자a 바로 뒤에 위치하는 선택자b를 선택(하나만 선택)
- 선택자a ~ 선택자b : 선택자a 뒤에 위치하는 모든 선택자b를 모두 선택
