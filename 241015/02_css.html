CSS : Cascading Style Sheests

* css 문법 
      선택자 { 속성 : 값; ..}
      selector { property : value; }
* css 적용 방식 3가지
  1. inline 방식 : 태그 옆에 직접적으로 부여하는 방식 (실무에서 거의 사용하지 않음)

    Inline 방식

  2. 내부 스타일 방식 : head 공간에 style 태그를 제작 후 css 문법을 작성해서 부여하는 방식 (실무에서 거의 사용하지 않음)

    내부 스타일 방식

  3. ★★★★★★
    외부 스타일 방식 : head 공간 안에 link 태그를 사용해서 .css 확장자를 연결 후 css 문서에 css 문법 작성해서 스타일 적용하는 방식 (실무에서 사용)
    <link rel="stylesheet" href="파일이름.확장자">
    rel : 문서 정보

    외부 스타일 방식

** 기초 선택자 **

전체 선택자
css : *
> html 내에 등장하는 모든 형태를 선택하는 선택자
> 초기화 값, 기본 세팅 값
태그 선택자
css : 태그 이름
> html 내에 등장하는 특정한 태그를 모두 선택하는 선택자
> 태그에 대한 기본 설정 값
아이디 선택자
css : #이름 / 태그 id="이름"
> 특정한 이름을 부여하여 선택하는 선택자
> html 화면 내에서 하나의 이름만 사용가능
> 주로 큰 단락, 부모박스에 많이 사용
> 클래스 선택자 보다 우선 순위
클래스 선택자
css : .이름 / 태그 class="이름"
> 특정한 이름을 부여하여 선택하는 선택자
> html 화면 내에서 여러 개의 이름을 사용가능
> 주로 작은 단락, 자식 박스에 많이 사용
> 변화가 잦은 요소
> 같은 속성의 스타일을 적용했을 때에는 제일 마지막에 쓴 최신 코드가 적용된다
