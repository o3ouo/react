글자 관련 속성

* color
> 글자 색상 : 단어, hex(컬러코드), rgb..
* font-size
> 글자 사이즈 단위 : px, em, rem, pt, v..
> html 기본 폰트 사이즈 : 16px
* font-weight
> 글자의 굵기 : lighter, normal, bold
> 100 ~ 900 (100~300 가늘게, 400~600 중간, 700~900 굵게)
* font-style
> 글자 스타일 : 이텔릭체 변경
* font-family
> 글꼴 : 내 컴퓨터에 저장된 글자가 표현
> 웹 폰트 제외
* letter-spacing
> 자간 : 글자 사이의 간격
> px, %, em..
* line-height
> 행간 : 줄간격 사이의 간격
> px, %, 배율
> 배율 단위로 많이 사용
* text-transform
> 글자 변형 : 대문자, 소문자, 첫 글자만 대문자
> uppercase : 모두 대문자로 변경
> lowercase : 모두 소문자로 변경
> capitalize : 띄어쓰기를 기준으로 단어의 첫 글자만 대문자로 변경
* writing-mode
글자 쓰기 방식 : 왼쪽에서 오른쪽, 위에서 아래로
Horizontal : 수평
Vertical : 수직
* text-align
> 글자 수평 정렬 : left, right, center
> 기본 값은 left
> 중간 정렬 시 많이 사용
* align-content
> 글자 수직 정렬 : start, center, and
> 해당 영역의 높이 값이 필수
* text-decoration
> 글자 선 : 선의 위치 (생략 불가능), 선의 종류 (생략 가능), 선의 색상 (생략 가능)
> 색상을 생략하면 폰트의 색상을 따라간다
> underline, overline, line-through

*** a 태그 사용 시 기본 설정 값 ***
* inherit
> 상위 요소의 설정을 따름
