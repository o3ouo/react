* transform: translate()
> 현재 위치를 기준으로 원하는 좌표의 값으로 축 이동할 때 사용
> (x, y)
> px, %, v.. (x, y축을 같이 써야 함)
> 하나의 값에 0을 주면 한 방향으로만 이동할 수 있음
* transform: translateX()
> 현재 위치를 기준으로 X축으로 이동할 때 사용
* transform: translateY()
> 현재 위치를 기준으로 Y축으로 이동할 때 사용

* transform: scale()
> 사이즈 확대, 축소
> (x, y)
> 1이 기준 값
> 배수값 0.9 < 1 < 1.1
* transform: scaleX()
> 사이즈를 X축으로 확대, 축소
* transform: scaleY()
> 사이즈를 Y축으로 확대, 축소
transform
* transform: rotate()
> 특정한 값 만큼 회전
> deg : 각도
> x, y 값을 둘 다 넣을 수 없고 하나의 값을 사용
> 시계 방향
> 음수의 값을 사용하면 반시계 방향
* transform: rotateX()
> X축을 기점으로 회전
* transform: rotateY()
> Y축을 기점으로 회전
> 움직이는 부모 박스에 perspective를 넣어서 원근감 표현
transform
* transform: skew()
> 특정한 각도로 기울이기, 왜곡
> deg : 각도
> x, y 값을 둘 다 넣으면 확인하기 힘듬
> 애니메이션화, 시각적 왜곡을 줄 때 사용
* transform: skew()
> X축으로 기울이기
* transform: skew()
> Y축으로 기울이기
