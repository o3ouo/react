table 속성

* border-collapse
> table과 td사이의 선 값의 유무를 결정하는 속성
> separate : 기본값. 선과 선 사이의 틈이 생긴 상태
> collapse : 선과 선사이의 틈을 병합 - table 표현시
> 부모에만 collapse 처리를 해도 충분히 결합이 된다
* border-spacing
> 셀들의 간격을 지정하는 값
> collapse 처리가 되어있지 않을 때 사용할 수 있다
> 1개 - 수평수직 / 2개 - 수평 수직
* empty-cells
> 빈셀의 보여지는 유무를 결정
> 기본값 : show
> table 태그에 부여
> hide를 지정하면 빈셀은 보이지 않는다
> collapse 처리가 되어있지 않을 때 사용할 수 있다
* vertical-align
> 수직에 대한 정렬
> middle : 기본값
> top, middle, bottom
> td 태그는 기본적으로 안에 있는 요소가 수직상 중심 배치가 된다
