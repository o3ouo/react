웹 폰트

webfont - 링크, 도메인, 브라우저와 상관없이 웹페이지 내에서 폰트가 연결되어지는 것

* TTF
> 마이크로소프트, 애플에서 협력해서 만듬
> 압축 파일
> 대부분 TTF 사용
> 높은 호환성
> 사파리에서 사용 x
* OTF
> TTF의 개선 버전
> 고해상도 출력 작업시 사용
> 어도비와 마이크로소프트에서 제작
> 대체문자 등 발전된 기능
> 인쇄물, 복잡한 그래픽 디자인에 사용
* WOFF/WOFF2
> 웹 사용에 최적화된 포맷
> 웹 전용
> 높은 압축률
> 높은 보안성
* EOT
> 인터넷 익스플로러에서 사용되었던 포맷

* import : 호출
> 필요한 폰트를 import에 한 번에 담아 호출해서 연결

** font-family : ''; **
내컴퓨터 내에서 설치된 폰트 연결

** CDN **
- 링크를 이용해서 서버상의 폰트를 가져와서 웹페이지에 전달하는 방법
- 느려지거나 연쇄 작용
1. 링크 태그 연결
2. import 연결

** @font-face **
- 링크를 이용해서 서버상의 폰트를 가져와서 웹페이지에 전달하는 방법
- 느려지거나 연쇄 작용
