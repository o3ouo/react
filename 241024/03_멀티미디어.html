오디오태그

: html5 에서 음악이나 오디오 같은 사운드를 재생하기 위해 사용, 태그 하나만으로 멀티미디어 기기에서 재생이 가능

* controls
> 사용자의 인터페이스를 제공하여 재생, 일시정지, 볼륨 등을 제어
> controls 속성이 반드시 있어야 사용할 수 있다
* loop
> 반복 재생
> 무한 재생
* muted
> 음소거 상태로 재생
* autoplay
> 페이지 로드 시 자동 재생
> 크롬, 사파리, 엣지 등에서는 자체적으로 자동 재생을 제한하고 있음
> 사용자가 오디오를 사용하겠다고 설정을 해둬야 하고, 정말 쓰고 싶다면 스크립트로 넣어야함
* volume
> 볼륨 조절 (0.0 ~ 1.0)
> 이렇게 사용하는 것보다 스크립트로 volume-min / volume-max 등으로 조절하는 것이 있음

비디오태그

: html5에서 영상을 실행시키기 위해 사용. 비디오와 같은 영상 스트리밍을 재생할 때 사용

* controls
> 사용자의 인터페이스를 제공하여 재생, 일시정지, 볼륨 등을 제어
> controls 속성이 반드시 있어야 사용할 수 있다
> controls 속성이 없을 경우 시작화면이 이미지처럼 나타남
* loop
> 반복 재생
> 무한 재생
* muted
> 음소거 상태로 재생
* autoplay
> 오디오와 다르게 음소거가 되어있을 경우 페이지 로드 시 자동 재생을 사용할 수 있음
> 사파리에서 영상 자동재생이 되지 않음
* playsinline
> 전체화면이 아닌 인라인으로 재생이 되도록 만드는 것이 목적
> 모바일 기기에서 재생 > ios 에서 autoplay가 제대로 사용되게 만듬
> ios에서 자동재생 시키기 위해서 사용
> 시작점 초기화 후 세팅
* poster
> 영상이 로드 되기 전 사용자가 재생버튼 누르기 전까지 표시할 이미지 지정
> 썸네일

iframe 태그

: 내부프레임 / html 문서 안에 다른 html 문서를 삽입할 수 있는 기능
: 유튜브 동영상, 구글 지도 등
: inline 요소라서 display : block 처리를 하고 사용

* titlt
> 제목 부여
